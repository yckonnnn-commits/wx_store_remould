# API 配置说明

## 阿里通义千问 API 配置

### 1. 获取 API Key

1. 访问阿里云控制台：https://dashscope.console.aliyun.com/
2. 登录后进入 API-KEY 管理页面
3. 创建新的 API Key 或使用现有的

### 2. 配置 API Key

打开配置文件：`config/model_settings.json`

找到 `阿里千问` 配置项，填入你的 API Key：

```json
"阿里千问": {
  "base_url": "https://dashscope.aliyuncs.com",
  "api_key": "YOUR_API_KEY_HERE",  // 在这里填入你的 API Key
  "model": "qwen-plus"
}
```

### 3. 保存并重启程序

配置完成后，保存文件并重启程序即可使用。

## 工作流程

1. **启动 AI** - 点击"启动 AI"按钮
2. **自动检测未读** - 程序每4秒检测一次未读消息
3. **自动进入聊天** - 检测到未读消息后自动点击进入
4. **抓取聊天记录** - 自动抓取完整聊天记录
5. **调用大模型** - 将聊天记录发送给阿里千问
6. **生成回复** - 大模型生成专业回复
7. **自动发送** - 将回复自动发送给用户

## 其他模型配置

如需使用其他模型（ChatGPT、DeepSeek、Kimi等），请在配置文件中填入对应的 API Key，然后在界面上切换模型即可。
